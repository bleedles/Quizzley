Creating a Quiz

Question Numbers

Question numbers are automatically updated whenever you add or remove a new question!

Choices

When adding choices for a question, they must be comma delimited (separated by commas). This is allows what you type to be broken up into mutliple choices and it's important for matching up the answer to the correct choice.

Answer

When setting the answer for a question, input the choice number with the first choice being 0, second choice being 1, etc.
